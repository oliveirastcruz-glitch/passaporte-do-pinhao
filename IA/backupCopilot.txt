✅ RESUMO DO PROJETO: PASSAPORTE DO PINHÃO
🧠 Objetivo
Criar um site regional para venda de cupons gastronômicos em Lages e região, conectando turistas e moradores com restaurantes parceiros que valorizam o pinhão e a cultura serrana.

🧠 main.py — Lógica do servidor
Cria o banco banco.db com a tabela empresas

Rota / → página inicial com botões

Rota /cadastro → formulário para cadastrar empresa

Rota /login → autenticação por email e senha

Rota /empresas → lista todas as empresas cadastradas

O servidor roda com python main.py e responde em http://127.0.0.1:5000/

🎨 index.html — Página inicial
Título e navegação com botões:

Cadastrar Empresa

Login

Ver Empresas

Seção de boas-vindas

Div reservada para futuras ofertas

📝 cadastro_empresa.html — Formulário de cadastro
Campos:

nome, email, senha, empresa, documento, telefone, foto

🔐 login.html — Formulário de login
Campos:

email, senha

Se o login falhar, redireciona para /cadastro

🧾 Banco de dados: banco.db
Tabela: empresas

sql
CREATE TABLE empresas (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    nome TEXT NOT NULL,
    email TEXT NOT NULL,
    senha TEXT NOT NULL,
    empresa TEXT NOT NULL,
    documento TEXT NOT NULL,
    telefone TEXT NOT NULL,
    foto TEXT
)
